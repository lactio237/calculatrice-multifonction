# Calculatrice Multifonction

Calculatrice interactive avec interface moderne, historique persistant et support des raccourcis clavier.

## 🎯 Fonctionnalités

- **Opérations basiques** : Addition, soustraction, multiplication, division
- **Fonctions avancées** : Pourcentage (%), inverse du signe (±), suppression (⌫)
- **Historique persistant** : Sauvegarde automatique dans localStorage
- **Replay** : Cliquez sur un calcul précédent pour le relancer
- **Raccourcis clavier** :
  - Chiffres : 0-9
  - Opérateurs : +, -, *, /
  - Entrée / = : Calculer le résultat
  - Backspace : Supprimer le dernier chiffre
  - Échap : Effacer tout
